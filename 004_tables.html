Hi Azu! In this lesson we will learn how to create HTML tables. Tables are a popular way to present information: they have a series of rows where each row is composed by one or more columns of information. An HTML table is created using the tag table. Tables usually have:
  • an optional caption that tells the reader what thwe table is about. It is created with the tag captiom
  • an optional header row where each column has a label that allows the reader to know what that column is about. It is created with the tag thead
  • a body composed by all rows of information that the table will show. It is created with the tag tbody
  • an optional footer row where each column usually summarizes the information of that column. It is created with the tag tfoot
Each row of information in the header, body, and footer is created with the tag tr. Each column of information inside a row is created with either the label th (short for table header, which makes the information be written in bolder font) or with the label td (short for table data, which is the normal one), as decided by the programmer creating the HTML table. Let's look at the following example table: 
<table>                                                                   
<caption>Population of the planets of the Solar System</caption>    
<thead>                                                                   
        <th>Name of the planet</th>                                 
        <th>Year 2021</th>                                          
        <th>Year 2022</th>                                          
        <th>The future</th>                                         
</thead>                                                                  
<tbody>                                                                   
        <tr>                                                              
                <td>Mercury</td>                                    
                <td>20</td>                                         
                <td>21</td>                                         
                <td>22</td>                                         
        </tr>                                                             
        <tr>                                                              
                <td>Venus</td>                                      
                <td>50</td>                                         
                <td>Muchos</td>                                     
                <td>Pocos</td>                                      
        </tr>                                                             
        <tr>                                                              
                <td>Earth</td>                                      
                <td>30</td>                                         
                <td>9 billion</td>                                  
                <td>Only fish</td>                                  
        </tr>                                                             
</tbody>                                                                  
<tfoot>                                                                   
        <tr>                                                              
                <th>Total:</th>                                     
                <th>80</th>                                         
                <th>A lot</th>                                      
                <th>A little</th>                                   
        </tr>                                                             
</tfoot>                                                                  
</table>
EXERCISES:
  1. Read the above HTML code and try to figure out how it should be represented in the browser.
  2. Write the above HTML code in a file, and open it with the browser. Observe how the fields are displayed. The columns are probably too tightly spaced, and the separation of the caption with respect to the table is probably too short; no lines are usually present sepeating the header from the data rows, and from the footer. All these presentation elements (lines, separations) will be controlled using the CSS language, which we will see in another lesson.
  3. Write a sample HTML page with a table that lists, for every UE country, its name, its capital, its population, and a popular dish.
